Chat Not Configured

The AI chat feature requires an LLM provider to be configured. Set the following environment variables to enable it:

LLM_API_KEY Your API key for the LLM provider Required for cloud providers
LLM_BASE_URL API endpoint URL (default: https://api.openai.com/v1) Optional
LLM_MODEL Model name to use (default: gpt-4o-mini) Optional

Local Ollama: No API key needed — just set LLM_BASE_URL and LLM_MODEL

Example Configurations

export LLM_API_KEY="sk-..."
export LLM_MODEL="gpt-4o"  # optional
export LLM_BASE_URL="http://localhost:11434/v1"
export LLM_MODEL="llama3"
export LLM_BASE_URL="https://api.together.xyz/v1"
export LLM_API_KEY="your-together-key"
export LLM_MODEL="meta-llama/Llama-3-70b-chat-hf"
